Welcome to Scrappy!

Welcome to Scrappy, the client-sided only web scraper!
Here is a quick rundown what each tab does!

Search:

This is to scrape HTML objects and text based on your search criteria!
The HTML Parser is very similar to JQuery Syntax. Use the Cheerio API to reference the selectors.
Use the textboxs to fill out what you want to find.
Here are given examples (without the quotation):

Finding a tag:
Ex.) "p", "br", "h1", "h2", "h3", etc...

Finding a certain class:
Ex.) ".my-class-name", ".findMeThisClass", etc...

Finding attributes in a tag:
This is abit tricky. If you get undefined, this means that no such attribute does not exist or such a attribute has not been set within the tag. Refer to JQuery use of attr().

Ex.) Suppose we have the following html tag:

[class="hello" tabid="goodbye" style="..." ...]

First inputbox --> .hello
Second inputbox (the attribute text box) --> tabid

This will return "goodbye"

Common Elements:

***Scrapping May take time due to multiple HTML Elements for longer webpages! ***

This is to scrape HTML objects that are commonly found amongst webpages. Here is a rundown list of all the tags and their attributes that we can examine:

Links:
1.) [a][href="..."]
2.) [links][href="..."]

Text:
1.) [span]
2.) [p]
3.) [h1, h2, h3, h4, h5, h6]

Navigate to the Common Elements and hit "Scrap!" to scrap the current URL that your DOM is focusing on (i.e the tab you are on)!
